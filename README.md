## 功能技术
* 用户注册
* 登录登出
* 功能菜单
* Dashboard
* 人员管理
* 菜单管理
* 我的信息
* 图片上传
* 视频上传
* 国际化
* 语言切换
* 主题切换
* Json转Dart Model类
* JWT

## flutter 版本
* 1.17.0

## 安装
```bash
flutter packages get
flutter create .
```

## 运行
```bash
flutter run -d chrome
```
