## 功能技术
* 登录登出
* 功能菜单
* 注册
* Dashboard
* 人员管理
* 我的信息
* 主题切换

* Json转Dart Model类
* JWT

## 安装
```bash
flutter packages get
flutter create .
```

## 运行
```bash
flutter run -d chrome
```
